5.1 - Linear functions

What is linear Functions? in short, A linear function is a basic type of function that has a constant rate of change. This means that for every equal change in the input (independent variable), there is a corresponding equal change in the output (dependent variable).

Here are some key points about linear functions:

  • Their graphs are straight lines on the coordinate plane.
  • They can be expressed in slope-intercept form as y = mx + b, where:
  • m is the slope, which represents the rate of change. It tells you how much the output changes for every unit change in the input.
  • b is the y-intercept, which represents the point where the line crosses the y-axis.

Representing Linear functions in notation form.

  • Function notation: F(x) = mx + b
  • Equation form: Y = mx + b
  • X: input value.
  • B: initial value of the dependent variable.
  • M: Equation form - Y = F(x) - Function notation
  • The rate of change or slope is over time so linear or constant that is the "M"
